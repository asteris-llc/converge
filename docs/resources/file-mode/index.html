file.mode

Mode monitors the mode of a file

Example

param "filename" {
  default = "test.txt"
}

file.mode "render" {
  destination = "{{param `filename`}}"
  mode        = 0777
}

Parameters

  • destination (string)

Destination specifies which file will be modified by this resource. The file must exist on the system (for example, having been created with file.content.)

  • mode (octal string)

Mode is the mode of the file, specified in octal.
